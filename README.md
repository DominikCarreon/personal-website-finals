Project Overview
This web application is your custom-built personal portfolio developed using the Vue JS framework. The project serves as an immersion into web architecture. It demonstrates your proficiency in full-stack frontend development, UI/UX design, and database programming. It functions as a dynamic application capable of real-time communication and data management.

Technical Stack
• Frontend Framework: Vue JS, HTML, CSS.
• Backend Database: Supabase for real-time data handling.
• Design: Figma and Canva.
• Architecture: Single-page application structure with modular sections.

Core Features & Functionalities
• Home and Profile Screen: Features a responsive profile layout. It includes personal branding, a bio, and a dynamic typing effect.
• Mission Log Filtering: A dynamic gallery sorting your recent operations. Visitors click buttons to filter projects by software and hardware categories. Examples include your digital PWD ID system and your Arduino fire detector.
• Secure Commlink Guestbook: An interactive module for sending messages. Changes sync in real-time with your Supabase database.
• Admin Command Center: A hidden authentication portal. You log in to access your database interface to delete spam.
• Active Radar Tracking: A live page view counter synced with the backend to display website traffic.
• Server Status Indicator: A real-time blinking ping indicator verifying database connectivity.
• Theme Management: Includes a toggle switch to change between dark mode and light mode.

Project Structure
The source code is organized to ensure readability.
• src/App.vue: The entry point containing the main logic, UI sections, Supabase configuration, and state management.
• src/style.css: Manages app-wide settings including the dark mode toggle and CSS variables.
• src/assets: Stores all optimized project images and profile pictures.

Implementation Highlights
• Database Integration: Successfully integrated Supabase to handle guestbook messages and view counts.
• Admin Authentication: Added a secure login route to manage database records directly from the frontend.
• Dynamic Rendering: Uses Vue JS reactive variables to ensure the user interface updates instantaneously.
